# F1-ChinaGP-Ticket-Monitor
A high-performance Java-based monitoring tool for F1 Chinese Grand Prix tickets with DingTalk notifications.
# F1 中国大奖赛门票实时监控系统

### 📌 项目简介
本项目是一款基于 **Java** 开发的实时票务监控工具，专门针对 F1 中国大奖赛官方门票库存进行自动化监控，解决人工刷票效率低、信息滞后的问题。

### 🚀 技术亮点
- **高频并发轮询：** 利用 Java 多线程技术实现对多个看台接口的并发监控，确保在门票库存变动的毫秒级时间内捕捉到信息。
- **即时预警推送：** 深度集成 **钉钉机器人 (DingTalk Webhook API)**，一旦发现库存状态从“缺货”变为“有票”，立即向移动端推送精准预警消息。
- **稳定性设计：** 针对票务系统的高频访问限制，设计了智能请求频率控制与异常重试机制，确保程序在无人值守状态下 7×24 小时稳定运行。
